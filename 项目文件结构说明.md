# 手写日记 - 项目文件结构说明

## 项目概述

「手写日记」是一款**水彩可爱笔记本风格**的个人日记应用，基于 Flutter 开发，支持 Android、iOS、Windows、macOS、Linux、Web 全平台。

**技术栈：** Flutter 3.x + Dart + Provider + SQLite

**UI 风格：** 水彩纹理背景 + 低饱和可爱配色 + 圆角卡片 + 柔和阴影的沉浸式手帐风格

**当前版本：** v2.9.0 (2026-06-09)

## 一、项目根目录文件

### 配置文件

| 文件 | 说明 |
|------|------|
| `pubspec.yaml` | **项目配置清单** - 定义项目名称、版本、Dart SDK 版本约束、所有依赖库、资源路径（背景图、字体、装饰图）、应用图标配置 |
| `pubspec.lock` | **依赖锁定文件** - 自动生成，锁定所有依赖的精确版本 |
| `analysis_options.yaml` | **代码分析配置** - 定义 Dart 静态分析规则 |
| `.gitignore` | **Git 忽略规则** - 排除构建产物、IDE 配置、临时文件 |
| `diary_app.iml` | **IntelliJ 项目文件** - IDE 项目配置 |

| 文件 | 说明 |

## 二、lib/ 源代码目录结构

### 2.1 入口文件

```
lib/
├── main.dart           # 应用入口
└── app.dart            # 应用根组件
```

#### main.dart
- **功能：** 应用启动入口
- **职责：**
  - 初始化 Flutter 绑定
  - 设置系统状态栏样式
  - 配置 Provider 状态管理（11 个 Provider）
  - 启动 MaterialApp

#### app.dart
- **功能：** 应用根组件和主外壳
- **职责：**
  - 定义 MaterialApp 配置（主题、路由）
  - 实现 MainShell 主界面外壳
  - 首次启动检测（欢迎页）
  - 水彩可爱风格底部导航栏（日记/成就/回顾/设置 + 中间写日记按钮）

## 八、UI 风格说明

### 水彩可爱笔记本风格

- **配色**：低饱和粉色/蓝色/绿色/黄色/紫色，柔和温馨
- **背景**：水彩纹理图片（季节切换）+ 水彩装饰元素（花朵/云朵/星星）
- **卡片**：大圆角（20px）、柔和阴影、半透明效果
- **字体**：MaShanZheng 手写体（标题）+ ZCOOLXiaoWei（正文）
- **装饰**：水彩花朵、云朵、星星、蛋糕、茶杯等 PNG 元素

### 季节主题

- 节气主题开启：使用节气对应的背景图（lichun.jpg 等 24 张）
- 节气主题关闭：使用水彩背景图（wc_bg_spring/summer/autumn/winter.png）
